웹 개발

웹 개발에 관한 풍부한 자료를 모아둔 개인 GitHub 저장소입니다. 최신 프론트엔드 및 백엔드 기술, 프레임워크, 디자인 패턴, 그리고 유용한 개발 도구 등에 대한 소스 코드와 학습 자료를 공유하고 있습니다. 웹 개발 분야에서의 지식 확장과 프로젝트에 도움이 될 자료들을 자유롭게 활용해보세요!

음악

음악 MIDI에 관한 다양한 자료를 모아둔 MIDI 관련 GitHub 저장소입니다. MIDI 프로토콜에 대한 이해부터 음악 작곡 및 프로덕션에 활용되는 MIDI 파일 예시, 유틸리티 및 라이브러리까지 폭넓게 다루고 있습니다. MIDI에 관심 있는 분들은 자료를 참고하여 음악 창작과 관련된 프로젝트에 활용해보세요!
